## About

Self-taught AI engineer and security researcher building things that shouldn't work -- but do.

By day I architect autonomous AI agent swarms on Kubernetes. By night I'm hunting bugs on HackerOne and designing open-source chip design tools. Everything runs on a 4-node K3s cluster with 9 named AI agents (inspired by Irish mythology), open-source models, and free-tier infrastructure. No VC money. No cloud bill. Just hardware, obsession, and a lot of terminal tabs.
